[풀스택]

- 상호명: 동네 옷가게
- 기술 스택: Next.js (풀스택), Vite, TailwindCSS, Zustand, Tanstack Query, SQLite(DB)

- 메인 페이지: 
	- 신상품, 추천 상품, 베스트셀러
- 상품 목록 페이지:	
	- 카테고리: 상의 (티셔츠, 셔츠, 트레이닝, 자켓), 하의 (슬랙스, 청바지, 트레이닝)
        - 정렬: 가격, 리뷰 개수, 판매량 (높은, 낮은 순)
- 상품 상세 페이지: 
	- 상품 이미지: 좌측에 배치
	- 상품 이름: 우측 상단에 배치
	- 사이즈: 상품 이름 아래에 배치
	- 색상 선택: 사이즈 아래에 배치
	- 수량: 색상 선택 아래에 배치
	- 가격: 수량 아래에 배치
	- 장바구니 담기 버튼: 가격 아래에 배치
	- 리뷰: 제품 이미지 장바구니 및 섹션에 배치
	- 설명: 리뷰 하단 섹션 컨텐츠 영역에 배치, 재질, 사이즈, 장점 등 자세한 설명
- 장바구니 페이지: 
	- 담은 상품 확인 하고 수량을 조절하며, 구매할 물품을 선택해서 결제할 수 있고, 총 주문 금액을 계산해서 보여준다.
- 결제 페이지: 
	- 배송지 정보 입력 (최근 배송지, 회원 정보 배송지, 새로운 배송지)
	- 주소 관리 기능
	- 결제 수단 선택
	- 결제 금액 표시
	- 결제 버튼
	
- 회원가입 페이지:
	- 아이디 (이메일 형식), 이름(한글), 비밀번호 (6자 이상), 비밀번호 확인 (비밀번호와 일치), 연락처(000-0000-0000 형식), 배송지 주소
- 로그인 페이지:
	- 아이디, 비밀번호, 로그인 버튼, 회원가입 버튼, 비밀번호 찾기 링크
- 마이 페이지:
	- 주문 내역 조회, 배송 상태 조회, 회원 정보 수정, 주소 관리
- 헤더:
	- 사이트 이름, 검색 버튼, 마이 페이지 버튼, 장바구니 버튼
- 푸터:
	- 보편적으로 많이 사용하는 것들로 구성
